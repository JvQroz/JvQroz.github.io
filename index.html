MINHAS TECNOLOGIAS:

HTML, que significa HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a principal linguagem usada para criar e estruturar conteúdos na web. Ele fornece a estrutura básica das páginas da web, definindo elementos como títulos, parágrafos, links, imagens e outros tipos de conteúdo.

CSS, que significa Cascading Style Sheets (Folhas de Estilo em Cascata), é uma linguagem usada para descrever a apresentação de um documento escrito em HTML. Enquanto o HTML define a estrutura e o conteúdo de uma página web, o CSS é responsável por sua aparência visual.

JavaScript é uma linguagem de programação essencial para o desenvolvimento web moderno. Ela é usada para adicionar interatividade e funcionalidades dinâmicas às páginas da web. Enquanto HTML fornece a estrutura e o CSS cuida da apresentação, o JavaScript permite que você manipule e interaja com o conteúdo da página de forma dinâmica.

React é uma biblioteca JavaScript de código aberto desenvolvida pelo Facebook para criar interfaces de usuário (UIs) interativas e dinâmicas, especialmente para aplicações web de uma única página (Single Page Applications - SPAs). Desde seu lançamento em 2013, React se tornou uma das ferramentas mais populares no desenvolvimento front-end.
